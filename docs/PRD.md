# IMPACT MATCH — MVP 產品需求文檔（PRD）

## 1. 產品概述

### 1.1 產品名稱
IMPACT MATCH

### 1.2 產品定位
香港影響力匹配平台，幫助企業按需求匹配社企產品、服務及工作坊，並將影響力數據化生成報告。

### 1.3 目標用戶
- **主要用戶**：香港企業 HR、採購部門、CSR/ESG 負責人
- **次要用戶**：平台管理員（管理社企資料）

### 1.4 高頻場景
- 員工福利（禮品、活動、工作坊）
- 企業採購（禮品採購、供應商合作）
- ESG 合作（社會效益項目配對）

---

## 2. MVP 範圍

### 2.1 In-Scope（MVP 包含）
1. 企業用戶註冊 / 登入（JWT 認證）
2. 企業填寫需求表單（類別、預算、人數、偏好標籤）
3. 系統推薦匹配的社企產品 / 服務 / 工作坊
4. 社企資料展示（名稱、類別、標籤、價格、影響力指標）
5. 影響力數據追蹤（受益人數、社會效益指標）
6. 生成影響力報告（PDF）
7. 管理員後台（添加 / 編輯社企資料）

### 2.2 Out-of-Scope（MVP 不包含）
- 線上支付 / 交易功能
- 社企端獨立帳號管理
- 即時聊天 / 配對後溝通
- AI 智能匹配（先用規則引擎）
- 多語言支持（先中文）
- 第三方登入（Google / Facebook）

## 4. 功能列表

### 4.1 認證模組
- 企業用戶註冊（公司名稱、聯絡人、電郵、密碼）
- 企業用戶登入（電郵 + 密碼 → JWT Token）
- JWT 驗證中間件

### 4.2 需求模組
- 創建需求（類別、預算範圍、人數、偏好標籤多選）
- 查看歷史需求
- 需求狀態管理（草稿 / 已提交 / 已匹配）

### 4.3 匹配模組
- 基於標籤加權的規則引擎匹配
- 匹配結果排序（相關度分數）
- 匹配結果分頁展示

### 4.4 社企展示模組
- 社企列表（卡片式）
- 社企詳情頁（產品/服務/工作坊、價格、影響力指標）
- 標籤篩選

### 4.5 影響力追蹤模組
- 記錄合作項目的受益人數
- 記錄社會效益指標（如：減少碳排放、創造就業、技能培訓人次）
- 影響力數據儀表板

### 4.6 報告模組
- 基於模板生成影響力報告
- 報告包含：企業資訊、合作項目列表、影響力數據匯總、圖表
- 導出 PDF

### 4.7 管理員後台
- 社企資料 CRUD
- 標籤管理
- 用戶需求列表（只讀）

## 5. 頁面流程

```
[首頁]
  ├── [註冊頁] → [登入頁]
  ├── [登入頁] ──┬── [企業儀表板]
                │       ├── [填寫需求] → [匹配結果]
                │       ├── [歷史需求] → [匹配結果]
                │       ├── [影響力追蹤] → [生成報告]
                │       └── [報告下載]
                └── [管理員後台]
                        ├── [社企管理]
                        ├── [標籤管理]
                        └── [需求列表]
```

### 5.1 頁面清單

| 頁面 | 路徑 | 說明 |
|------|------|------|
| 首頁 | / | 平台介紹、登入/註冊入口 |
| 註冊 | /register | 企業用戶註冊 |
| 登入 | /login | 企業用戶/管理員登入 |
| 企業儀表板 | /dashboard | 概覽：最新需求、推薦、影響力數據 |
| 填寫需求 | /demand/new | 多步驟表單 |
| 匹配結果 | /matches | 推薦社企列表 |
| 社企詳情 | /social-enterprise/:id | 社企詳細資料 |
| 影響力追蹤 | /impact | 數據儀表板 |
| 報告頁 | /reports | 報告列表與生成 |
| 管理員後台 | /admin | 社企管理、標籤管理 |

## 6. 非功能需求

### 6.1 性能
- 頁面加載 < 2 秒
- 匹配算法響應 < 500ms（1000 條社企資料內）

### 6.2 安全
- 密碼 bcrypt 加密存儲
- JWT 過期時間 24 小時
- API 速率限制（100 次/分鐘）

### 6.3 兼容性
- 現代瀏覽器（Chrome、Firefox、Safari、Edge 最新兩版）
- 響應式設計（桌面優先，兼容平板）
